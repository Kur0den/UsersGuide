# BFJ-Documents

これはDiscordコミュニティー 【BattlefieldJapanCommunity】の公式ドキュメントです。
Discord初心者へのドキュメントでもあり、利用者に対してのドキュメントでもあります。

## このドキュメントへの協力
このドキュメントは公開されており、ドキュメント編集は鯖民が参加することが出来ます。
詳しくは [こちら](/DocumentsAuthor/how-to.md) をご覧ください。
